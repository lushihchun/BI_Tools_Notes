## Metabase 開源版
- 建構邏輯  
  以「提問」為基本單位，一個提問內含一串資料處理過程及一份產出的視覺化圖表，所有資料的串階及處理皆係於各個提問下獨立進行。  
  儀錶板中排入各個「提問(圖表)」後，由使用者自行在儀表板介面中，手動設定各圖表或篩選器內資料欄的關係，達到互動篩選效果。
- 資料
    - 可接受來源  
      MySQL、PostgreSQL、SQLServer、Amazon Redshift、BigQuery、Snowflake、Amazon Athena、ClickHouse、Detabricks、Druid、DruidJDBC、MongoDB、Presto、SparkSQL、SQLite、Starburst
    - 串接&新增欄位功能  
        - 內建基礎資料處理功能，可用點選方式進行資料表的欄位串接、自訂欄位及篩選，並儲存於提問中。
        - 自訂欄位部分，提供數十個基礎函式協助資料處理。
        - 資料「模型」功能：(未使用過，應該是設定固定資料處理程序之功能。)
    - 語言  
      使用繁體中文未遇困難。
- 圖表
    - 圖表種類：基本款完備  
        - 表格、柱狀圖(直/橫)、折線圖、圓餅圖、面積圖、組合圖、地圖、散點圖、瀑布圖、數字卡、透視表、趨勢數字卡、量測計卡、進度卡(達標標示)、漏斗圖、桑基圖
    - 圖內運算  
        - 總行數、總和、平均、中位數、不重複值總數、累積和、累積行數、標準差、最小值、最大值
        - 在特定圖表中可自行計算百分比(eg.堆疊百分比長條圖、圓餅圖等)
    - 圖表可調整性：中等  
        - 可調整顯示與否，但無法調整細節。
        - 可在既有色卡內自行調整顏色配置。
    - 地圖  
        - 可使用geojson自定義地圖，惟須為網路連結，並建立於系統管理者權限下。
        - 地圖檔中須包含geojson區塊及區塊標籤。每張地圖僅能選一欄標籤作為對外串接之依據。
        - 只能以數值(5階顏色深淺)描述區域地圖。

    - 互動

    - 中文介面
