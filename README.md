## Metabase
- 建構邏輯  
  以圖表(提問)為基本單位，資料的串階及處理係於每張圖表下各自進行。  
  儀錶板中排入各種圖表後，使用者自行在儀表板介面中設定各圖表或篩選器中資料欄的關係。
- 資料
    - 可用來源
    - 串接功能
    - 新增欄位功能
- 圖表
    - 圖表種類
    - 圖內運算
    - 內容可調整姓
    - 語言
    - 地圖
    - 操作
- 儀錶板
    - 排版
    - 互動
- 專案
    - 元素整理
